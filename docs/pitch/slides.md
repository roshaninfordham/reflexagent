# What Reflex is

```
Autonomous monitor (OpenFDA / RSS, every 60s)
                ↓
      [novel signal]  →  fires the swarm WITHOUT a human click
                ↓
  Eleven specialized agents (3 tiers, parallel where possible)
  ┌─────────────┬──────────────────────────────┬─────────────┐
  │  Ingest     │   Decision                    │  Synthesis  │
  ├─────────────┼──────────────────────────────┼─────────────┤
  │  Inbound    │  Triage                      │  Writer     │
  │  Scout      │  Verify + Counter (adversarial) │ Auditor   │
  │  Recon      │  Cohort                       │  Publisher  │
  │             │  Substitute (BioNeMo ESM2)    │             │
  │             │  Routing + Comms              │             │
  └─────────────┴──────────────────────────────┴─────────────┘
                ↓
   Published, cited, routed — visible on the open web
```

# The trick: the counter-evidence agent

Every recall is cross-referenced through **two passes**:

1. **Confirmation** — claims supported by ≥2 independent sources from FDA / EMA / PubMed.
2. **Adversarial** — actively hunts manufacturer rebuttals, sponsored studies, regulator clarifications.

If a meaningful contradiction surfaces, the workflow's verdict becomes **`requires_human`** — never silently broadcast.

<br/>

> Live: the Apotex investor-relations statement contradicting the FDA's NDMA finding for metformin → flagged in red on the canvas → held for review.

# NVIDIA BioNeMo · drug substitution

When a drug is recalled, **`Substitute`** ranks alternatives by molecular target similarity.

```
Recalled: Metformin → target PRKAA1 (AMPK α1)
                          ↓
NIM Llama 3.3 70B picks 3 candidates
                          ↓
ESM2-650M embeds each target protein → 1280-d vectors
                          ↓
Cosine similarity ranking
                          ↓
1. Sitagliptin (DPP4)    sim 0.933  ← top
2. Glipizide   (KCNJ11)  sim 0.895
3. Pioglitazone (PPARG)  sim 0.000  ← off-target, ranked last
```

Brief renders the 2D chemical structure + 3D protein cartoon for each.
